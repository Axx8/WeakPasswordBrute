# WeakPasswordBrute
 一款针对弱口令进行批量扫描爆破的工具，目前支持FTP | Redis | Mysql | SSH | SQL Server   持续更新中。

### 运行
```
python3 Weak_Password_Windows.py  ip.txt
```
![Weak_Password_Linux](/images/Weak_Password_Windows.png)

### Linux运行
```
./Weak_Password_Linux  ip.txt
```
